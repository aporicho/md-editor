# 步骤

> 对文档的最小原子操作。每个步骤描述一次具体的修改——替换内容、添加标记、修改属性等。步骤可以被应用、反转、序列化。

## Step trait

所有步骤类型实现同一个 trait：

| 方法 | 说明 |
|------|------|
| step.apply(doc) → StepResult | 应用到文档，返回成功（新文档）或失败（错误信息） |
| step.invert(doc) → Step | 基于原始文档生成逆向步骤（撤销用） |
| step.get_map() → StepMap | 返回该步骤产生的位置映射 |
| step.map(mapping) → Option\<Step\> | 将步骤中的位置通过映射器转换（协同编辑 rebase 用）。映射后可能无效则返回 None |
| step.merge(other) → Option\<Step\> | 尝试将两个连续的同类型步骤合并为一个（压缩历史用） |

---

## StepResult

步骤应用的结果。步骤不一定能成功——比如替换后可能违反约束器规则。

| 变体 | 说明 |
|------|------|
| Ok(doc) | 成功，包含修改后的新文档 |
| Err(message) | 失败，包含错误描述 |

---

## 步骤类型详解

### ReplaceStep

最基础也是最常用的步骤。用切片替换文档中 from 到 to 的范围。

| 字段 | 说明 |
|------|------|
| from | 替换范围的起始位置 |
| to | 替换范围的结束位置 |
| slice | 用来替换的切片 |
| structure | 是否强制结构匹配（阻止不合法的开放深度） |

**几乎所有编辑操作都归结为 ReplaceStep：**

| 操作 | ReplaceStep |
|------|-------------|
| 删除 | from=5, to=7, slice=空 |
| 插入文字 | from=5, to=5, slice=包含文本的切片 |
| 替换选区 | from=选区开始, to=选区结束, slice=新内容 |
| 回车换段 | from=光标位置, to=光标位置, slice=包含两个开放段落的切片 |
| 粘贴 | from=光标位置, to=光标位置, slice=剪贴板内容 |

#### 回车换段的切片

回车在光标位置拆分段落，本质上是插入一个"两段都开放"的切片：

```
光标在 paragraph 内的位置 7:

slice.content = Fragment [paragraph[], paragraph[]]
slice.open_start = 1    ← 第一个 paragraph 是开放的，融入前半段
slice.open_end = 1      ← 第二个 paragraph 是开放的，融入后半段

效果: 一个段落变成两个段落，光标在第二个段落开头
```

#### invert 示例

```
原始文档: "一段粗体文本"

步骤: ReplaceStep(from=5, to=7, slice=Slice::empty())  // 删除 "粗体"
逆向: ReplaceStep(from=5, to=5, slice=Slice(Fragment[text("粗体")[bold]], 0, 0))  // 把 "粗体" 插回去

逆向步骤需要原始文档来获取被删除的内容，所以 invert 接收 doc 参数。
```

### ReplaceAroundStep

替换范围的"外壳"但保留中间的内容。用于改变节点的包裹层级而不动内部内容。

| 字段 | 说明 |
|------|------|
| from | 外部范围起始 |
| to | 外部范围结束 |
| gap_from | 保留内容的起始 |
| gap_to | 保留内容的结束 |
| slice | 新的外壳切片 |
| insert | 保留内容插入到切片中的位置 |
| structure | 是否强制结构匹配 |

```
把 paragraph 包裹进 blockquote:

  from=0, to=14 (整个 paragraph 含开闭标签)
  gap_from=1, gap_to=13 (paragraph 内部内容)
  slice = blockquote 的开闭标签 + 内部的 paragraph 开闭标签

  效果: 外壳从 <p>...</p> 变成 <blockquote><p>...</p></blockquote>
        中间的文本内容不动
```

典型用途：

| 操作 | 说明 |
|------|------|
| wrap | 给节点套一层（段落 → 引用块中的段落） |
| lift | 把节点提升一级（引用块中的段落 → 段落） |
| 修改块类型 | 把 paragraph 变成 heading（替换外壳标签） |

---

### AddMarkStep / RemoveMarkStep

给指定范围的行内内容添加或移除标记。

| 字段 | 说明 |
|------|------|
| from | 范围起始 |
| to | 范围结束 |
| mark | 要添加/移除的标记 |

内部逻辑：遍历范围内的所有文本节点，逐个修改标记集。会自动处理文本节点的拆分和合并。

```
给位置 5~9 的 "粗体文本" 添加 bold:

原始: paragraph [text("一段粗体文本。")]
步骤: AddMarkStep(from=5, to=9, mark=bold)
结果: paragraph [text("一段"), text("粗体文本")[bold], text("。")]

文本节点被拆分为三段，中间一段带上了 bold 标记。
```

---

### AddNodeMarkStep / RemoveNodeMarkStep

给指定位置的整个节点（而非范围内的文本）添加或移除标记。

| 字段 | 说明 |
|------|------|
| pos | 节点位置 |
| mark | 要添加/移除的标记 |

用于给 image 等行内叶子节点设置标记（比如给图片加链接）。

---

### AttrStep

修改指定位置节点的单个属性。

| 字段 | 说明 |
|------|------|
| pos | 节点位置 |
| attr | 属性名 |
| value | 新属性值 |

```
把一级标题改成二级标题:
  AttrStep(pos=0, attr="level", value=2)
```
