# 事务

> 继承自变换器，在步骤序列之上额外追踪选区变化、暂存标记和元数据。是编辑器状态更新的唯一途径。

## 与变换器的关系

```
变换器 (Transform)
  ├── doc, steps, docs, mapping    ← 变换器的字段
  ├── replace, delete, insert ...  ← 变换器的方法
  │
  └── 事务 (Transaction) extends 变换器
        ├── selection              ← 新增：选区追踪
        ├── stored_marks           ← 新增：暂存标记
        ├── meta                   ← 新增：元数据
        └── set_selection, ...     ← 新增：状态相关方法
```

事务 = 变换器 + 状态关注的东西。你在变换器上能做的操作（delete、insert、add_mark 等）在事务上都能做。

---

## 字段

继承自变换器的字段（doc、steps、docs、mapping）之外：

| 字段 | 说明 |
|------|------|
| cur_selection | 事务中设置的新选区（未设置时为初始选区） |
| cur_selection_for | 标记选区是在哪个步骤之后设置的（用于自动映射） |
| stored_marks | 暂存标记 |
| meta | 元数据键值对 |
| time | 事务创建的时间戳 |
| updated | 位标记，记录哪些状态字段被显式修改过（选区、暂存标记等） |

---

## 选区相关方法

| 方法 | 说明 |
|------|------|
| tr.set_selection(selection) | 显式设置新选区 |
| tr.replace_selection(slice) | 用切片替换当前选区范围 |
| tr.replace_selection_with(node) | 用节点替换当前选区范围 |
| tr.delete_selection() | 删除选区内容 |
| tr.selection | 获取当前选区（如果事务中有步骤，会自动映射） |

### 选区的自动映射

事务中的步骤会改变文档，选区位置也需要跟着变。如果没有显式调用 set_selection，事务会自动将选区通过映射器转换到新位置。

```
光标在位置 10，事务删除了位置 4~6:

tr.delete(4, 6)
tr.selection.from → 8    （自动从 10 映射到 8，减了 2）
```

如果在步骤之后显式设置了选区，则使用显式值，不再自动映射：

```
tr.delete(4, 6)
tr.set_selection(TextSelection::create(tr.doc, 0))  // 光标移到文档开头
```

## 暂存标记方法

| 方法 | 说明 |
|------|------|
| tr.set_stored_marks(marks) | 设置暂存标记 |
| tr.ensure_marks(marks) | 仅当暂存标记与 marks 不同时才设置（避免无意义的更新） |
| tr.add_stored_mark(mark) | 向暂存标记中添加一个标记 |
| tr.remove_stored_mark(mark) | 从暂存标记中移除一个标记 |

---

## 元数据 (Meta)

事务可以携带键值对形式的元数据，用于组件间通信：

| 方法 | 说明 |
|------|------|
| tr.set_meta(key, value) | 设置元数据 |
| tr.get_meta(key) → Option\<Value\> | 读取元数据 |

### 常用元数据

| 键 | 值 | 说明 |
|----|----|----|
| "addToHistory" | false | 告诉历史插件不要记录这个事务（光标移动、选区变化不需要撤销） |
| "paste" | true | 标记这是一个粘贴操作 |
| "pointer" | true | 标记这是一个鼠标操作（区分键盘和鼠标） |
| 插件键 | 任意值 | 插件之间的自定义通信 |

插件应该用自己的插件键作为元数据键，避免字符串键冲突。

---

## 典型事务示例

### 用户输入文字

```
let tr = state.tr();
tr.replace_selection_with(schema.text("a", state.stored_marks_or_cursor_marks()));
// 选区自动更新到插入文字之后
```

### 用户按 Ctrl+B 切换粗体

```
// 有选区: 给选区内容添加/移除 bold 标记
let tr = state.tr();
if 选区内已有 bold:
    tr.remove_mark(from, to, bold)
else:
    tr.add_mark(from, to, bold)

// 无选区（光标）: 切换暂存标记
let tr = state.tr();
if stored_marks 中有 bold:
    tr.remove_stored_mark(bold)
else:
    tr.add_stored_mark(bold)
```

### 用户按回车

```
let tr = state.tr();
tr.split(cursor_pos);
// split 内部: 在光标位置插入两个开放段落的切片
// 选区自动映射到第二个段落的开头
```
