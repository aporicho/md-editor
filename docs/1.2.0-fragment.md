# 片段

> 节点子节点的不可变有序序列。每个节点都持有一个片段作为子节点容器，无子节点时为空片段。

## 结构

片段内部就是一个 `Vec<Node>` 的不可变包装。

```
paragraph 的片段:
┌──────────────────────────────────────┐
│ text("一段")  text("粗体")[bold]  text("文本。") │
└──────────────────────────────────────┘
```

```
horizontal_rule 的片段:
┌──┐
│  │  ← 空片段，叶子节点没有子节点
└──┘
```

---

## 关键属性

| 属性 | 说明 |
|------|------|
| size | 所有子节点大小之和（用于位置计算） |
| child_count | 子节点数量 |

---

## 关键操作

| 操作 | 说明 |
|------|------|
| fragment.child(index) | 取第 index 个子节点 |
| fragment.maybe_child(index) | 同上，越界返回 None |
| fragment.for_each(f) | 遍历所有子节点 |
| fragment.cut(from, to) | 按位置截取子片段，返回新片段 |
| fragment.insert_at(pos, fragment) | 在指定位置插入片段，返回新片段 |
| fragment.replace_child(index, node) | 替换第 index 个子节点，返回新片段 |
| fragment.append(other) | 拼接两个片段，返回新片段。相邻的同标记文本节点会自动合并 |
| fragment.text_content() | 拼接所有文本内容 |
| fragment.find_index(pos) | 按位置找到对应的子节点索引和节点内偏移 |
| fragment.find_diff_start(other) | 找到两个片段第一个不同的位置 |
| fragment.find_diff_end(other) | 找到两个片段最后一个不同的位置 |
| fragment.eq(other) | 结构相等比较 |
| Fragment::from(nodes) | 从节点数组创建片段 |
| Fragment::empty() | 空片段单例 |

---

## 文本合并

片段保证相邻的同标记文本节点总是被合并。例如：

```
append 之前:
  片段A: [text("一段")]
  片段B: [text("文本")]

append 之后:
  [text("一段文本")]    ← 自动合并为一个文本节点
```

但标记不同时不会合并：

```
  片段A: [text("一段")]
  片段B: [text("粗体")[bold]]

append 之后:
  [text("一段"), text("粗体")[bold]]    ← 保持分开
```

这个不变量简化了后续所有操作——不需要处理"两个相邻的纯文本节点"这种冗余情况。
