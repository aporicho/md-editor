# 变换器

> 步骤序列的高层构建器。提供面向编辑操作的方法（删除、插入、添加标记、拆分节点等），内部自动分解为步骤、维护映射器、更新文档。

## 结构

| 字段 | 说明 |
|------|------|
| doc | 当前文档（随每个步骤应用而更新） |
| steps | 已应用的步骤序列 |
| docs | 每个步骤应用前的文档快照序列（steps[i] 应用于 docs[i] 得到 docs[i+1]） |
| mapping | 所有步骤的累积映射器 |
| before | 变换开始前的原始文档（= docs[0]） |

---

## 方法分类

### 内容操作

| 方法 | 说明 |
|------|------|
| tr.replace(from, to, slice) | 用切片替换范围。最底层的内容操作，其他方法大多基于它。 |
| tr.replace_with(from, to, content) | 用节点或片段替换范围。封装 replace，自动构造切片。 |
| tr.delete(from, to) | 删除范围。等价于 replace(from, to, Slice::empty())。 |
| tr.insert(pos, content) | 在位置插入节点或片段。等价于 replace(pos, pos, ...)。 |

### 标记操作

| 方法 | 说明 |
|------|------|
| tr.add_mark(from, to, mark) | 给范围内的行内内容添加标记。内部生成 AddMarkStep。 |
| tr.remove_mark(from, to, mark) | 移除范围内的指定标记。内部生成 RemoveMarkStep。 |
| tr.clear_incompatible(pos, type) | 清除指定位置节点中与目标类型不兼容的内容和标记。用于块类型转换前的准备。 |

### 节点操作

| 方法 | 说明 |
|------|------|
| tr.set_node_markup(pos, type, attrs, marks) | 修改节点类型、属性、标记。用于块类型转换（如段落 → 标题）。 |
| tr.set_node_attribute(pos, attr, value) | 修改节点的单个属性。内部生成 AttrStep。 |

### 结构操作

| 方法 | 说明 |
|------|------|
| tr.split(pos, depth) | 在位置拆分节点。depth 控制拆分的层数（默认 1 = 拆分最内层）。 |
| tr.join(pos) | 合并 pos 两侧的相邻节点。pos 必须在两个兄弟节点之间。 |
| tr.lift(range, target) | 将范围内的内容提升到 target 深度的祖先层级。（取消嵌套） |
| tr.wrap(range, wrappers) | 用新节点层级包裹范围内的内容。（增加嵌套） |

## split 详解

回车换行的核心操作。在光标位置拆分段落：

```
文档: doc > paragraph > [text("一段文本")]
光标在 "段" 和 "文" 之间（位置 3）

tr.split(3)

结果: doc > paragraph[text("一段")] > paragraph[text("文本")]
```

depth 参数控制拆分深度：

```
文档: doc > blockquote > paragraph > [text("内容")]
光标在 "内" 和 "容" 之间

tr.split(pos, 1):  拆分 paragraph，blockquote 不变
  doc > blockquote > paragraph["内"] > paragraph["容"]

tr.split(pos, 2):  拆分 paragraph 和 blockquote
  doc > blockquote[paragraph["内"]] > blockquote[paragraph["容"]]
```

## lift 和 wrap 详解

### lift — 提升（取消嵌套）

```
文档: doc > blockquote > paragraph["内容"]

tr.lift(range, target=0)

结果: doc > paragraph["内容"]
```

paragraph 从 blockquote 中提升到 doc 层级。内部用 ReplaceAroundStep 去掉 blockquote 的外壳。

### wrap — 包裹（增加嵌套）

```
文档: doc > paragraph["内容"]

tr.wrap(range, [{type: blockquote}])

结果: doc > blockquote > paragraph["内容"]
```

内部用 ReplaceAroundStep 在 paragraph 外面套上 blockquote。

wrappers 是数组，可以一次套多层：

```
tr.wrap(range, [{type: bullet_list}, {type: list_item}])

结果: doc > bullet_list > list_item > paragraph["内容"]
```

## 步骤的自动位置转换

变换器中后续步骤使用的位置是**基于当前文档**的，不需要手动映射：

```
let mut tr = Transform::new(doc);

// 原始文档中位置 4~6 有 "CD"
tr.delete(4, 6);
// 删除后文档变了，现在位置 4 后面是 "E"

// 这里的 4 是删除后的文档中的位置，不需要手动从 6 映射
tr.insert(4, schema.text("X"));
```

但如果你持有的是**原始文档的位置**，需要通过映射器转换：

```
let old_pos = 8;  // 原始文档中的位置
let new_pos = tr.mapping.map(old_pos);  // 映射到当前文档中的位置
```

## 与事务的关系

事务 (Transaction) 继承自变换器，额外追踪：
- 选区变化
- 暂存标记
- 元数据（是否加入历史等）

所有变换器的方法在事务上都可以直接使用。
